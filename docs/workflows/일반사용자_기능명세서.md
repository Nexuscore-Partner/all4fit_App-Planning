# 일반 사용자 기능 명세서

## 1. 인증 및 접근

### 1.1 로그인

|--------|------|------|-------------|----------|

| 전화번호 로그인 | 전화번호/비밀번호 로그인 | 전화번호 로그인 폼 | 로그인 메인 → 전화번호 입력 → 비밀번호 입력 → 로그인 | 로그인 중 |

| 비밀번호 찾기 | 이메일/전화번호로 비밀번호 재설정 | 비밀번호 찾기 폼 | 로그인 메인 → 비밀번호 찾기 → 이메일/SMS 발송 | 재설정 중 |
| 비밀번호 변경 | 현재 비밀번호 확인 후 변경 | 비밀번호 변경 폼 | 마이페이지 → 설정 → 비밀번호 변경 | 변경 중 |

## 7. 권한 및 제약사항

### 7.1 일반 사용자 권한
- 시설 정보 조회 가능
- 예약 신청 가능
- 리뷰 작성/조회 가능
- 커뮤니티 참여 가능
- 1:1 문의 가능
- 즐겨찾기 관리 가능

### 7.2 제약사항
- 시설 정보 수정 불가
- 다른 사용자 문의 답변 불가
- 관리자 기능 접근 불가
- 지도자/운영자 전용 기능 사용 불가
