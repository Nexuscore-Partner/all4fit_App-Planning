# 역할별 화면 UI 설계 및 워크플로우 (1차 개발)

## 1. 일반 사용자 화면 UI 설계

### 1.1 로그인/회원가입 화면
```
┌─────────────────────────────────────┐
│           모두의 핏 로고             │
├─────────────────────────────────────┤
│  [이메일 로그인] [전화번호 로그인]    │
│  [카카오 로그인] [구글 로그인]       │
├─────────────────────────────────────┤
│           [회원가입]                │
├─────────────────────────────────────┤
│        [비밀번호 찾기]              │
└─────────────────────────────────────┘
```

**워크플로우:**
1. 앱 실행 → 로그인 화면 표시
2. 로그인 방식 선택 → 해당 로그인 폼 표시
3. 회원가입 선택 → 사용자 타입 선택 화면
4. 비밀번호 찾기 → 이메일/SMS 인증 화면

### 1.2 메인 화면 (홈 탭)
```
┌─────────────────────────────────────┐
│  📍 현재 위치: 강남구 역삼동  [변경]  │
├─────────────────────────────────────┤
│  🔍 [시설 검색창]                   │
├─────────────────────────────────────┤
│  🏃‍♂️ 운동 분야 선택                │
│  [구기] [투기/무예] [육상/트랙]     │
│  [수영] [기타]                      │
├─────────────────────────────────────┤
│  📍 근처 인기 시설                  │
│  ┌─────┐ ┌─────┐ ┌─────┐           │
│  │축구장│ │농구장│ │테니스│           │
│  └─────┘ └─────┘ └─────┘           │
└─────────────────────────────────────┘
```

**워크플로우:**
1. 로그인 완료 → 메인 화면 표시
2. GPS 위치 자동 인식 → 현재 위치 표시
3. 운동 분야 선택 → 세부 종목 선택 화면
4. 시설 검색 → 검색 결과 화면

## 4. 공통 워크플로우 (1차 개발)

### 4.1 로그인 프로세스
```
앱 실행 → 로그인 화면 → 로그인 방식 선택 → 인증 → 메인 화면
```

### 4.2 회원가입 프로세스
```
로그인 화면 → 회원가입 → 사용자 타입 선택 → 정보 입력 → 관리자 승인 → 계정 활성화
```

### 4.3 시설 조회 프로세스 (일반 사용자)
```
시설 검색 → 시설 선택 → 시설 상세 → 리뷰 조회 → 즐겨찾기 추가
```

### 4.4 문의 프로세스
```
일반 사용자: 마이페이지 → 1:1 문의 → 문의 작성 → 문의 접수
지도자: 문의 관리 → 문의 선택 → 답변 작성 → 답변 등록
```

### 4.5 알림 프로세스
```
이벤트 발생 → 서버 처리 → 푸시 알림 발송 → 앱에서 알림 수신 → 사용자 확인
```

## 5. 화면 전환 플로우 (1차 개발)

### 5.1 일반 사용자 화면 전환
```
로그인 → 메인(홈) → 시설 검색 → 시설 상세 → 리뷰 조회 → 마이페이지
```

### 5.2 시설 운영자 화면 전환
```
로그인 → 운영자 메인 → 시설 관리 → 리뷰 관리 → 고객 관리
```

### 5.3 지도자 화면 전환
```
로그인 → 지도자 메인 → 문의 답변 → 프로필 관리
```

## 6. 권한별 접근 제어 (1차 개발)

### 6.1 화면 접근 권한
- **일반 사용자**: 기본 기능 + 시설 조회 + 리뷰 + 커뮤니티 + 1:1 문의
- **시설 운영자**: 일반 사용자 기능 + 시설 관리 + 리뷰 관리 + 고객 관리
- **지도자**: 일반 사용자 기능 + 1:1 문의 답변 + 프로필 관리

### 6.2 기능 제한
- 시설 정보 수정: 시설 운영자만 가능
- 1:1 문의 답변: 지도자만 가능
- 관리자 기능: 관리자만 접근 가능
- 예약 기능: 2차 개발에서 추가 예정

## 7. 1차 개발 핵심 기능

### 7.1 일반 사용자
- 시설 검색 및 조회
- 시설 상세 정보 확인
- 리뷰 작성 및 조회
- 즐겨찾기 관리
- 1:1 문의 작성
- 커뮤니티 참여

### 7.2 시설 운영자

- 리뷰 관리 및 답글
- 고객 관리
- 시설 운영 통계

### 7.3 지도자
- 1:1 문의 답변
- 지도자 프로필 관리
- 답변 내역 관리
- 답변 통계 확인
