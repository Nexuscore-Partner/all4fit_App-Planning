- 레슨 수익 조회 가능

### 9.2 제약사항
- 시설 정보 수정 불가
- 다른 지도자의 레슨 관리 불가
- 관리자 기능 접근 불가
- 일반 사용자 문의만 답변 가능 (시설 운영자 문의 답변 불가)
