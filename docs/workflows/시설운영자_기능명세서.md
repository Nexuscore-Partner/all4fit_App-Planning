## 9. 권한 및 제약사항

### 9.1 시설 운영자 권한
- 자신이 등록한 시설 정보 수정 가능
- 자신의 시설 예약 관리 가능
- 자신의 시설 리뷰 관리 가능
- 매출 현황 조회 가능
- 고객 관리 가능
- 운영자 전용 문의 가능

### 9.2 제약사항
- 다른 운영자의 시설 정보 수정 불가
- 지도자 전용 기능 사용 불가
- 관리자 기능 접근 불가
- 일반 사용자 문의 답변 불가 (지도자만 가능)
