## 💡 포트폴리오 강점

이 프로젝트를 통해 보여줄 수 있는 것:

✅ **체계적인 문서화 능력**
   - 6,000+ 라인의 상세 기획 문서
   - 역할별 기능명세서 (4개)
   - 완전한 API 설계 (50+ 엔드포인트)

✅ **시스템 설계 능력**
   - 멀티 롤 아키텍처 설계
   - 데이터 모델링 (MongoDB)
   - RESTful API 설계

✅ **사용자 경험 설계**
   - 역할별 맞춤 기능 정의
   - 통합 회원가입 플로우
   - IA 및 네비게이션 설계

✅ **기술 이해도**
   - React Native 모바일 개발
   - Node.js 백엔드 아키텍처
   - JWT 인증, 소셜 로그인

✅ **프로젝트 관리**
   - 3단계 개발 로드맵
   - 우선순위 설정
   - 명확한 스코프 정의
