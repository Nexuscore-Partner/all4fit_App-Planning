## 6. 권한 및 제약사항

### 6.1 지도자 권한
- 1:1 문의 답변 가능
- 지도자 프로필 관리 가능
- 답변 내역 관리 가능
- 답변 통계 조회 가능
- 일반 사용자 문의만 답변 가능

### 6.2 제약사항
- 시설 정보 수정 불가
- 다른 지도자의 문의 답변 불가
- 관리자 기능 접근 불가
- 시설 운영자 문의 답변 불가
- 레슨 관리 기능 사용 불가 (2차 개발에서 추가 예정)
- 고객 관리 기능 사용 불가 (2차 개발에서 추가 예정)

## 7. 1차 개발 핵심 기능

### 7.1 지도자 핵심 기능
- **1:1 문의 답변**: 일반 사용자의 문의에 대한 답변 작성
- **프로필 관리**: 지도자 정보, 자격증, 포트폴리오 관리
- **답변 관리**: 답변 내역, 통계, 품질 관리
- **알림 관리**: 문의 알림, 시스템 알림 수신

### 7.2 제외된 기능 (2차 개발에서 추가 예정)
- 레슨 관리
- 고객 관리
- 지도자 커뮤니티
- 수익 관리
- 일정 관리
