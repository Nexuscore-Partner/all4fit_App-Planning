## 🏗️ 시스템 아키텍처

### 플랫폼 구성
```
모두의 핏 시스템
├── 모바일 앱 (React Native)
│   ├── 일반 사용자
│   ├── 시설 운영자
│   └── 지도자
└── 웹 시스템 (React)
    └── 관리자
```

### 주요 기능
- **인증 시스템**: 통합 로그인, 소셜 로그인, 비밀번호 찾기
- **시설 관리**: 검색, 상세 정보, 즐겨찾기
- **리뷰 시스템**: 리뷰 작성, 관리, 신고
- **커뮤니티**: 게시판, 댓글, 좋아요
- **문의 시스템**: 1:1 문의, 답변 관리
- **관리자 시스템**: 사용자 관리, 콘텐츠 관리, 시스템 관리

## 🔧 기술 스택

### Frontend
- **모바일**: React Native
- **웹**: React
- **상태관리**: Redux Toolkit
- **네비게이션**: React Navigation

### Backend
- **서버**: Node.js + Express
- **데이터베이스**: MongoDB
- **인증**: JWT + OAuth
- **파일업로드**: Multer

### 외부 서비스
- **지도**: Kakao Maps API / Naver Maps API
- **푸시알림**: Firebase Cloud Messaging
- **소셜로그인**: 카카오 API
