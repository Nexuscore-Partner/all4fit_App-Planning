# 1차 개발 통합 기능 명세서 요약

## 📋 개요
모두의 핏 앱 1차 개발을 위한 역할별 기능 명세서 통합 요약

## 👥 역할별 핵심 기능

### 1. 일반 사용자
| 카테고리 | 기능 | 설명 | 상태 |
|----------|------|------|------|
| **인증** | 로그인/회원가입 | 이메일, 전화번호, 소셜 로그인 | ✅ |
| **시설 조회** | 시설 검색/상세 | 위치 기반 시설 검색 및 상세 정보 | ✅ |
| **리뷰** | 리뷰 작성/조회 | 시설 리뷰 작성 및 조회 | ✅ |
| **즐겨찾기** | 즐겨찾기 관리 | 시설 즐겨찾기 추가/삭제 | ✅ |
| **문의** | 1:1 문의 | 고객센터 문의 작성 | ✅ |
| **커뮤니티** | 게시판/댓글/공감 | 에브리타임 스타일 게시판 (글쓰기, 댓글, 공감) | ✅ |

### 2. 시설 운영자
| 카테고리 | 기능 | 설명 | 상태 |
|----------|------|------|------|
| **인증** | 운영자 가입/승인 | 사업자 정보 검증 후 승인 | ✅ |
| **시설 관리** | 시설 정보 수정 | 시설명, 주소, 운영시간, 요금 등 | ✅ |
| **리뷰 관리** | 리뷰 답글/신고 | 시설 리뷰에 답글 작성 및 신고 처리 | ✅ |
| **고객 관리** | 고객 현황/소통 | 고객 목록 조회 및 메시지 발송 | ✅ |
| **통계** | 운영 통계 | 시설 조회, 리뷰 통계 등 | ✅ |

### 3. 지도자
| 카테고리 | 기능 | 설명 | 상태 |
|----------|------|------|------|
| **인증** | 지도자 가입/승인 | 자격증 검증 후 승인 | ✅ |
| **프로필 관리** | 지도자 정보 | 자격증, 포트폴리오, 소개 관리 | ✅ |
| **문의 답변** | 1:1 문의 답변 | 일반 사용자 문의에 답변 작성 | ✅ |
| **답변 관리** | 답변 내역/통계 | 답변 이력 및 성과 관리 | ✅ |

### 4. 관리자
| 카테고리 | 기능 | 설명 | 상태 |
|----------|------|------|------|
| **사용자 관리** | 회원/지도자/운영자 | 모든 사용자 승인 및 관리 | ✅ |
| **콘텐츠 관리** | 리뷰/문의/신고 | 모든 콘텐츠 관리 및 처리 | ✅ |
| **시스템 관리** | 설정/데이터 | 시스템 설정 및 데이터 관리 | ✅ |
| **분석** | 통계/리포트 | 사용자 및 서비스 분석 | ✅ |

## 🔐 권한 체계

### 접근 권한
- **일반 사용자**: 기본 기능 + 시설 조회 + 리뷰 + 게시판 커뮤니티
- **시설 운영자**: 일반 사용자 기능 + 시설 관리 + 리뷰 관리
- **지도자**: 일반 사용자 기능 + 문의 답변 + 프로필 관리
- **관리자**: 모든 기능 접근 가능

### 기능 제한
- 시설 정보 수정: 시설 운영자만
- 1:1 문의 답변: 지도자만
- 사용자 승인: 관리자만
- 시스템 설정: 관리자만
